# Vector Enterprise – Automation Testing Using Selenium (Java) 

## 📌 Project Overview

Vector Enterprise is an automation testing project developed to validate the functionality of the Distribusync system.  
This project uses **Selenium with Java** to automate end-to-end testing scenarios, ensuring reliability, accuracy, and performance of the application.

The framework is designed to support scalable and maintainable automated test execution.
